CSS Cheat Sheet

This table lists 15 useful CSS elements, properties, and functionalities.

# Property / Feature Description Example
1 Selectors Targets specific HTML elements p, .class, #id
2 color Sets the text color color: red;
3 background Sets background color or image background: #eee;
4 font-family Sets the font of the text font-family: Arial;
5 font-size Sets the size of the text font-size: 16px;
6 font-weight Defines boldness of text font-weight: bold;
7 text-align Aligns text left, center, or right text-align: center;
8 margin Space outside the element margin: 20px;
9 padding Space inside the element padding: 10px;
10 border Adds a border around an element border: 1px solid black;
11 display Defines how an element is displayed display: flex;
12 flexbox Flexible layout control justify-content: space-between;
13 position Controls positioning of elements position: absolute;
14 animation / keyframes Adds animations to elements @keyframes slidein { ... }
15 media queries Makes site responsive @media (max-width: 600px) { ... }
