JavaScript Cheat Sheet

Here are 15 JavaScript concepts with examples to help you build dynamic and interactive web apps!

# Topic Description Example
1 Variables Store data 
let name = \"Carol\";
2 Data Types Different value types 
string, number, boolean
3 Functions Reusable code blocks 
function greet() { alert(\"Hi\"); }
4 If/Else Conditional logic 
if (score > 50) {...}
5 Loops Repeat actions 
for (let i = 0; i < 5; i++) {...}
6 Arrays Store multiple items 
let fruits = [\"apple\"];
7 Objects Key-value pairs 
{name: \"Carol\", age: 25}
8 Events User actions 
onclick=\"doSomething()\"
9 DOM Interact with HTML 
document.getElementById(...)
10 console.log() Debugging 
console.log(\"Hi\");
11 Math Math operations 
Math.random();
12 String Methods Work with text 
\"hello\".toUpperCase()
13 Array Methods Modify arrays 
fruits.push(\"banana\");
14 setTimeout Delay code 
setTimeout(() => {}, 1000);
15 Arrow Functions Short functions 
const hi = () => {}
