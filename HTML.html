HTML Cheat Sheet

This table lists 15 useful HTML elements and their purposes. Perfect for quick review!

Element Purpose Example
<h1> - <h6> Headings (largest to smallest) <h1>Title</h1>
<p> Paragraph of text <p>Hello world!</p>
<a> Hyperlink <a href=\"https://example.com\">Visit</a>
<img> Image <img src=\"img.jpg\" alt=\"Description\">
<ul>, <ol>, <li> Lists <ul><li>Item</li></ul>
<table> Table structure <table></table>
<tr>, <td>, <th> Table row, cell, header <tr><td>Data</td></tr>
<form> Form container <form action=\"\"></form>
<input> Form input field <input type=\"text\">
<button> Button element <button>Click Me</button>
<br> Line break Text<br>New Line
<hr> Horizontal line <hr>
<strong>, <em> Bold and italic text <strong>Bold</strong>
<div>, <span> Containers for layout and inline style <div>Block</div>
<header>, <footer>, <main> Semantic page layout <main>Content</main>
